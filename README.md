# 1. Project Overview

Plant Disease Detection is a computer vision based machine learning project that identifies diseases in plant leaves using image processing and deep learning techniques. The system analyzes leaf images and predicts whether the plant is healthy or infected with a specific disease.

This project helps farmers and agricultural experts detect plant diseases early, reducing crop loss and improving agricultural productivity.

# 2. Problem Statement

Plant diseases significantly affect crop yield and food production. Manual disease detection requires expert knowledge and is time-consuming.

The goal of this project is to build an automated system that can classify plant diseases from leaf images using machine learning/deep learning techniques.

# 3. Dataset

The project uses the PlantVillage dataset, which contains thousands of labeled plant leaf images.
Dataset features:
  50,000+ images
  38 different plant diseases
  Healthy and diseased leaves
Dataset columns:
  Image
  Plant type
  Disease label
Example classes:
  Tomato Early Blight
  Potato Late Blight
  Apple Scab
  Healthy Leaf
